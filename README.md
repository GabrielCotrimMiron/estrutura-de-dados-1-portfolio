# 📚 Portfólio - Estrutura de Dados I

## Sobre

Este repositório reúne os códigos desenvolvidos durante a disciplina **Estrutura de Dados I**, do curso de **Engenharia de Software**.

Os exemplos foram implementados em linguagem **C** e representam os principais conteúdos abordados ao longo das aulas.

---

## Conteúdos

| Arquivo | Conteúdo |
|---------|----------|
| 01_conceitos_basicos.c | Variáveis, operadores, estruturas de decisão e repetição |
| 02_ponteiros.c | Ponteiros |
| 03_structs.c | Structs |
| 04_vetores_e_matrizes.c | Vetores e Matrizes |
| 05_lista_encadeada.c | Lista Encadeada |
| 06_lista_circular.c | Lista Circular |
| 07_lista_duplamente_encadeada.c | Lista Duplamente Encadeada |
| 08_pilha.c | Pilhas (Stack) |
| 09_fila.c | Filas (Queue) |
